# Packeta Docs

This directory stores the raw documentation made available by the Packeta group.

Important to know:

- Packeta operates in multiple countries, including CZ/SK/PL/HU/RO.
- In Hungary, now the Packeta group owns Foxpost, which will bring some changes after January.
- The Packeta API can be used to ship to Foxpost Boxes as well.

Docs for their general multinational API can be found here: <https://docs.packeta.com/docs/introduction>
