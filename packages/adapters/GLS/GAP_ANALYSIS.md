## Service Codes (OpenAPI Appendix B + Service schema)

All 23 service codes are now supported. Services are built from two sources:

1. **Auto-derived** from parcel data:
   - `PSD` — injected automatically for PICKUP_POINT deliveries
   - `SAT` — injected when `options.gls.saturdayDelivery` is true
   - `T09`/`T10`/`T12` — express services mapped from `parcel.service = 'express'` or `parcel.service = 'overnight'`

2. **Explicit** via `options.gls.services` array:
   - Any service code with its parameter object

### Service Parameter Types

```ts

{ value: string }

// Decimal parameter (INS, MMP)
{ value: number }

// DateTime parameter (DDS)
{ value: string } // ISO 8601 date-time

// String + Decimal parameter (DPV)
{ stringValue: string, decimalValue: number }

// Time range parameter (SDS)

// ADR parameter (complex)
{ value: string } // ADR settings JSON
```
